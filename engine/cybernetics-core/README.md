# @cybernetics/core

TypeScript implementation of cybernetics primitives — Viable System Model, variety engines, algedonic signals, homeostats, heterarchy, and autopoiesis verification.

This library was written by the CynCo author as a TypeScript implementation of Stafford Beer's cybernetics concepts. The source comments reference "Rust cybernetics::* modules" — these refer to an earlier private Rust prototype by the same author that was used as the design reference for this TypeScript port.

## Concepts Implemented

- **Variety Engine** — Ashby's Law of Requisite Variety
- **Algedonic Channel** — Pain/pleasure signal routing with kill switch
- **VSM Node** — Beer's Viable System Model structure
- **Homeostat** — Coupled differential equations for system stability
- **Heterarchy** — Dynamic authority switching between systems
- **Observer Effects** — Measurement divergence tracking
- **Autopoiesis** — Self-production verification
- **Constraints** — Beer's constraint satisfaction
